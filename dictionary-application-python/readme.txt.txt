--------Welcome to the readme file of this dictionary application---------


-This is a dictionary app which runs on command line and is written in python 3.9.6

-It contains 2 files app-code.py and data.json

-app-code.py contains the code of this application which could be run using python interpreter.

-data.json file is a json file which contains the word meanings in key-value pairs and acts as a local database from which the application is fetching the word meanings for the user.

-made this project as part of my python learning journey.
